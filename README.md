# MobileNet-CIFAR10-Lightweight-Classification

## Project Overview
This project implements MobileNetV2 on the CIFAR-10 dataset for lightweight image classification using transfer learning and PyTorch.

## Features
- MobileNetV2 Transfer Learning
- CIFAR-10 Dataset
- GPU Training (CUDA)
- Custom Loss Function
- Confusion Matrix
- Classification Report
- Training Accuracy and Loss Curves

## Dataset
CIFAR-10 contains 60,000 images across 10 classes:
- Airplane
- Automobile
- Bird
- Cat
- Deer
- Dog
- Frog
- Horse
- Ship
- Truck
